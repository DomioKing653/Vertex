# Flare

This main git repo for flare programming language.

---

## Why to chose flare
- **Staticly typed**:catch all of the type errors at compile time
- **Fast compilation speed**:compile your programs in seconds
- **Compiled to bytecode**:You can see the exact code that the compiler will emit and save
---

## Staring with flare

For now flare can be build from source, be installed via our installer or can be installed from selected releases. For more info look at the **[Installing flare](installation/INSTALLATION.md)**.

When you are ready you may look at the **[Getting started](docs/GETTING_STARTED.md)**
